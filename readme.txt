Игра "100-клеточные шашки"
Автор: Широков Сергей (группа КН-202)

Описание:
Данная программа является реализацией игры "100-клеточные шашки".

Требования:
-Python не ниже версии Python 3.4
-PyQt5

Запуск:
./startMenu.py
(В открывшемся окне будет кнопка "Help", при нажатии на которую можно получить краткую инструкцию по тому, как играть)
